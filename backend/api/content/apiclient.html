About persistence

						
Run after receive response & Parameters

						You can add any valid JavaScript code to "Run after receive response" area.

						Use the global object $$ to save information.

						Use the global object json to read data from the response.

						
For example:

						
$$.myuser = json.user_id

						After posting the request, the code will run and you'll see the global parameters added to the "Parameters" section.


						In the request inputs, you can use any parameter from the global object
						
For example:

						
$$.myuser

						
Saved calls

						You can save request with their params and scripts, giving them names for later use.
